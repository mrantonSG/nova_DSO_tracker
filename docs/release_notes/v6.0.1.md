# Nova DSO Tracker v6.0.1

## Bug Fixes

- **ADMIN_USERS env var now honoured** - Fixed multi-user admin access respecting the `ADMIN_USERS` environment variable instead of being hardcoded to "admin".
- **Large journal photo archive imports** - Increased `MAX_CONTENT_LENGTH` to 500MB to support importing large journal photo archives.
- **Post-midnight dusk/sunset annotations** - Fixed mobile chart annotations handling post-midnight dusk and sunset times correctly.
- **Chart annotation timestamps** - Fixed chart annotations using Luxon timezone-aware milliseconds for accurate timestamp handling.
