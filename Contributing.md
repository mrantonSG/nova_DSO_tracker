# Contributing to Nova DSO Tracker

Thank you for considering a contribution to Nova DSO Tracker!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request with a clear description of what you changed and why

Please try to match the existing code style and structure of the project.

## Questions

If you have questions before starting work on a larger feature, feel free to open an issue first to discuss the idea.
