
#### Administrar mis objetos Esta es la sala de control central de su biblioteca de objetivos. Aquí puede refinar los detalles de los objetos, organizar sus proyectos y mantener limpia su base de datos.

**Filtrar su biblioteca** Si tiene una biblioteca grande, utilice la barra de filtro de la parte superior para encontrar rápidamente objetivos específicos. Puede filtrar por: * **ID o nombre:** (por ejemplo, «M42» u «Orion») * **Constelación o tipo:** (por ejemplo, «Cyg» o «Nebula»)
* **Estado:** filtre por objetos **habilitados** o **deshabilitados**, o vea qué elementos están **compartidos** o **importados**. * **Fuente:** busque objetos de catálogos específicos (por ejemplo, «messier»). Si desea seleccionar específicamente una sola fuente, utilice «=messier».

**Acciones de selección (operaciones masivas)** Utilice las casillas de verificación para seleccionar varios objetos y, a continuación, utilice los botones de acción: * **Habilitar/Deshabilitar:** Cambie el estado activo de los objetos seleccionados. Los objetos deshabilitados se omiten durante los cálculos para obtener un mejor rendimiento. * **Obtener detalles:** Consulte catálogos externos para completar los datos que faltan de los objetos seleccionados. Esto actualiza los campos **Tipo**, **Magnitud**, **Tamaño**, **Brillo superficial** y **Constelación** que están vacíos o muestran «N/A». * **Eliminar:** elimina permanentemente los objetos seleccionados. Los objetos con entradas de diario o proyectos no se pueden eliminar hasta que se eliminen primero.

**Desactivar objetos (rendimiento)** Para que la aplicación Nova siga funcionando rápidamente, puede **desactivar** los objetos que no está visualizando actualmente pero que no desea eliminar. * **¿Por qué desactivarlos?** Los objetos desactivados se omiten durante los cálculos pesados (como el panel de control y Outlook), lo que aumenta significativamente el rendimiento del software.
* **Diario seguro:** Puede desactivar un objeto de forma segura sin perder datos. Las entradas y registros existentes en el **Diario** para ese objeto seguirán siendo totalmente funcionales y visibles. * **Cómo hacerlo:** Puede alternar el estado **Activado/Desactivado** en un solo objeto, o utilizar la barra **Acciones de selección** en la parte superior para activar o desactivar muchos objetos a la vez.

**Edición de datos de objetos** Puede editar la mayoría de los campos directamente en la lista sin salir de la página: * **Coordenadas y estadísticas:** Las actualizaciones de RA/DEC, magnitud, tamaño o brillo superficial son útiles si desea refinar la precisión de la posición o añadir datos que faltan. * **Proyecto activo:** Marque esta casilla para marcar el objeto como prioritario. Los objetos «Los objetos «activos» se resaltan de forma especial en la herramienta de previsión **Outlook**. **Notas y uso compartido** * **Notas privadas:** utilice este espacio para ideas personales, planes de exposición o recordatorios. Estas notas nunca se comparten. * **Notas compartidas:** si marca **«Compartir este objeto»,** estas notas serán visibles para otros usuarios del servidor. **Guardar cambios**

* **Actualización única:** Haga clic en el botón **Actualizar** dentro del cuadro de un objeto para guardar los cambios solo para ese objetivo. * **Actualización masiva:** Haga clic en el botón grande **Actualizar objetos** en la parte inferior para guardar los cambios en toda la lista. * **Eliminación:** Marque la casilla **Eliminar** (o utilice el botón Eliminación masiva) para eliminar un objeto de forma permanente.
