# Comparación de objetos secundarios El menú desplegable **Añadir objeto** le permite comparar la curva de altitud de otro objeto junto con su objetivo principal en la vista gráfica. ## Cómo se clasifican los objetos Los objetos del menú desplegable se **ordenan por duración total observable** (el más largo primero). Esto le ayuda a identificar rápidamente qué objetivos activos serán visibles durante más tiempo esta noche.

## Criterios de filtrado Los objetos solo aparecen en la lista si cumplen **todos** los siguientes requisitos: 1. **Proyecto activo**: el objeto debe estar marcado como proyecto activo (casilla de verificación en la pestaña Notas y encuadre).
2. **Observable esta noche**: el objeto debe estar por encima de la máscara del horizonte durante la oscuridad astronómica. 3. **Coordenadas válidas**: el objeto debe tener definidas las coordenadas RA y DEC. 4. **No es el principal**: el objeto que está viendo actualmente se excluye de la lista. ## Límites

- Solo se muestran los **20 objetos principales** por duración observable. - Se excluyen los objetos con **0 minutos** de tiempo observable. ## Visualización - La altitud del objeto secundario se muestra como una **línea magenta continua**. - No se representa ninguna línea acimutal para el objeto secundario. (para reducir el desorden en el gráfico). - Al seleccionar **Ninguno**, se elimina la comparación secundaria y se restaura la vista predeterminada. ## Consejos - Utilice esta función para planificar su noche comparando varios objetivos. - Es ideal para decidir entre objetos que transitan en diferentes momentos. - El título de la página se actualiza para mostrar los nombres de ambos objetos cuando hay una comparación activa.
