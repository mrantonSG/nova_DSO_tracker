
#### Buscar y fusionar duplicados Con el tiempo, es posible que su biblioteca acumule entradas duplicadas para el mismo objetivo (por ejemplo, importó «M 42» manualmente, pero también añadió «NGC 1976» desde un catálogo). Esta herramienta analiza su base de datos para encontrar y resolver estos conflictos. **Cómo funciona**

La aplicación Nova analiza toda su biblioteca de objetos para encontrar pares de objetos que se encuentran a menos de **2,5 minutos de arco** entre sí. **Resolución de duplicados** Cuando se encuentran duplicados, se muestran uno al lado del otro. Debe decidir qué versión es la «maestra» (la que conserva) y cuál es el duplicado que se fusionará y eliminará.

* **Conservar A, fusionar B:** el objeto A se mantiene. El objeto B se elimina, pero sus datos se transfieren a A. * **Conservar B, fusionar A:** el objeto B se mantiene. El objeto A se elimina, pero sus datos se transfieren a B.

**¿Qué pasa con mis datos?** La fusión es un **proceso inteligente** diseñado para conservar su historial. Cuando fusiona un objeto: * **Diarios:** Todas las sesiones de imágenes vinculadas al objeto eliminado se vuelven a vincular al objeto «conservado». * **Proyectos:** Todos los proyectos activos o pasados se trasladan al objeto «conservado».
* **Encuadres:** Se transfieren los datos de encuadre guardados. (Nota: si *ambos* objetos tenían encuadres guardados, se conserva el encuadre del objeto «conservado»). * **Notas:** ¡Las notas privadas no se pierden! Las notas del objeto eliminado se añaden al final de las notas del objeto «conservado».

**Consejo:** Por lo general, es mejor conservar el objeto con el nombre más común (por ejemplo, conservar «M 31» y fusionar «Galaxia de Andrómeda») para facilitar las búsquedas futuras.
