
#### Importar desde el catálogo Esta pestaña le permite importar de forma masiva colecciones seleccionadas de objetos proporcionadas por el servidor. Esta es la forma más rápida de completar una nueva biblioteca. **Paquetes de catálogo** La tabla muestra los «paquetes» disponibles (por ejemplo, «Objetos Messier», «Catálogo Caldwell», «Catálogo Sharpless»). Cada entrada muestra:

* **Nombre y descripción:** lo que contiene el paquete. * **Etiquetas:** categorías rápidas (por ejemplo, «Nebulosa», «Principiante»). * **Recuento:** cuántos objetos hay en el paquete. **Cómo funciona la importación** Haga clic en el botón **Importar** junto a un paquete para comenzar.

* **Nuevos objetos:** Los objetos que aún no tiene se añadirán a su biblioteca. * **Objetos existentes (actualizaciones):** Si ya tiene un objeto (por ejemplo, «M42»), Nova **actualizará** sus campos «Inspiración» (URL de la imagen, descripción, créditos) con los datos de alta calidad del catálogo.
* **Seguro para datos personalizados:** sus notas de proyecto personales, estado activo y encuadres guardados **nunca se sobrescriben**. La importación solo mejora los datos estáticos del catálogo.
