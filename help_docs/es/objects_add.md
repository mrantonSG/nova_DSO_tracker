
#### Añadir nuevo objeto Utilice esta pestaña para ampliar su biblioteca. La **aplicación TERM_6** le ayuda a automatizar la introducción de datos utilizando bases de datos astronómicas profesionales. **Método 1: Búsqueda automática (recomendado)**

1. Introduzca un **ID de objeto** (por ejemplo, «M31», «NGC 7000», «Gaia 123...»). 2. Haga clic en **Buscar**. 3. Nova comprobará primero su biblioteca local. Si no lo encuentra, consultará la base de datos **SIMBAD**.
4. Si lo encuentra, los campos (RA, DEC, Magnitud, Tamaño, Tipo) se rellenarán automáticamente. 5. Haga clic en **Confirmar añadir** para guardarlo en su biblioteca. (También puede hacer clic en **editar** si desea cambiar el contenido rellenado automáticamente, por ejemplo, el nombre).

**Método 2: Entrada manual** Si desea añadir un objetivo que no se encuentra mediante la búsqueda automática: 1. Rellene el **ID del objeto** y el **Nombre**. 2. Escriba manualmente las coordenadas **RA** y **DEC**. 3. Rellene estadísticas opcionales como Magnitud o Constelación.
4. Haga clic en **Confirmar añadir**. **Añadir notas inmediatamente** Puede añadir sus **Notas privadas** (por ejemplo, «Plan de mosaico») o **Notas compartidas** aquí mismo antes de guardar, para que su objeto esté listo en el momento en que se añada.
