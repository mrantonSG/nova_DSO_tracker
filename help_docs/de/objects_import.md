
#### Aus Katalog importieren

Dieser Tab ermöglicht es Ihnen, kuratierte Objektsammlungen, die vom Server bereitgestellt werden, in Massen zu importieren. Dies ist der schnellste Weg, eine neue Bibliothek zu füllen.

**Katalog-Pakete**

Die Tabelle zeigt verfügbare "Pakete" an (z. B. "Messier-Objekte", "Caldwell-Katalog", "Sharpless-Katalog"). Jeder Eintrag zeigt:

* **Name & Beschreibung:** Was das Paket enthält.
* **Tags:** Schnelle Kategorien (z. B. "Nebel", "Anfänger").
* **Anzahl:** Wie viele Objekte im Paket enthalten sind.

**Wie der Import funktioniert**

Klicken Sie auf die Schaltfläche **Importieren** neben einem Paket, um zu beginnen.

* **Neue Objekte:** Objekte, die Sie derzeit nicht haben, werden zu Ihrer Bibliothek hinzugefügt.
* **Vorhandene Objekte (Updates):** Wenn Sie bereits ein Objekt haben (z. B. "M42"), wird Nova seine "Inspiration"-Felder (Bild-URL, Beschreibung, Credits) mit den hochwertigen Daten aus dem Katalog **aktualisieren**.
* **Sicher für benutzerdefinierte Daten:** Ihre persönlichen Projekt-Notizen, Aktiv-Status und gespeicherten Framings werden **niemals überschrieben**. Der Import erweitert nur die statischen Katalogdaten.
