
#### Meine Objekte verwalten

Dies ist die zentrale Steuerzentrale für Ihre Zielbibliothek. Hier können Sie Objektdetails verfeinern, Ihre Projekte organisieren und Ihre Datenbank sauber halten.

**Ihre Bibliothek filtern**

Wenn Sie eine große Bibliothek haben, verwenden Sie die Filterleiste oben, um spezifische Ziele schnell zu finden. Sie können filtern nach:

* **ID oder Name:** (z. B. "M42" oder "Orion")
* **Sternbild oder Typ:** (z. B. "Cyg" oder "Nebel")
* **Status:** Filtern nach **Aktivierten** oder **Deaktivierten** Objekten, oder sehen Sie, welche Elemente **Geteilt** oder **Importiert** sind.
* **Quelle:** Finden Sie Objekte aus bestimmten Katalogen (z. B. "messier"). Wenn Sie spezifisch nur eine Quelle auswählen möchten, verwenden Sie "=messier".

**Auswahlaktionen (Stapelverarbeitung)**

Verwenden Sie die Kontrollkästchen, um mehrere Objekte auszuwählen, und verwenden Sie dann die Aktionsschaltflächen:

* **Aktivieren / Deaktivieren:** Schalten Sie den Aktiv-Status ausgewählter Objekte um. Deaktivierte Objekte werden während der Berechnungen für bessere Leistung übersprungen.
* **Details abrufen:** Fragen Sie externe Kataloge ab, um fehlende Daten für ausgewählte Objekte zu ergänzen. Dies aktualisiert **Typ**, **Magnitude**, **Größe**, **Oberflächenhelligkeit** und **Sternbild**-Felder, die leer sind oder "N/A" anzeigen.
* **Löschen:** Entfernen Sie ausgewählte Objekte dauerhaft. Objekte mit Journal-Einträgen oder Projekten können nicht gelöscht werden, bis diese zuerst entfernt wurden.

**Objekte deaktivieren (Leistung)**

Um die Nova App schnell zu halten, können Sie Objekte **deaktivieren**, die Sie derzeit nicht fotografieren, aber nicht löschen möchten.

* **Warum deaktivieren?** Deaktivierte Objekte werden während intensiver Berechnungen (wie Dashboard und Ausblick) übersprungen, was die Softwareleistung erheblich erhöht.
* **Journal-sicher:** Sie können ein Objekt sicher deaktivieren, ohne Daten zu verlieren. Ihre vorhandenen **Journal**-Einträge und Protokolle für dieses Objekt bleiben voll funktionsfähig und sichtbar.
* **Wie:** Sie können den **Aktiviert/Deaktiviert**-Status bei einem einzelnen Objekt umschalten oder die **Auswahlaktionen**-Leiste oben verwenden, um viele Objekte gleichzeitig zu aktivieren oder deaktivieren.

**Objektdaten bearbeiten**

Sie können die meisten Felder direkt in der Liste bearbeiten, ohne die Seite zu verlassen:

* **Koordinaten & Statistiken:** Updates zu RA/DEC, Magnitude, Größe oder Oberflächenhelligkeit sind nützlich, wenn Sie die Positionsgenauigkeit verbessern oder fehlende Daten hinzufügen möchten.
* **Aktives Projekt:** Aktivieren Sie dieses Kontrollkästchen, um das Objekt als Priorität zu markieren. "Aktive" Objekte erhalten eine besondere Hervorhebung im **Ausblick**-Prognosetool.

**Notizen & Teilen**

* **Private Notizen:** Verwenden Sie diesen Bereich für persönliche Framing-Ideen, Belichtungspläne oder Erinnerungen. Diese werden nie geteilt.
* **Geteilte Notizen:** Wenn Sie **"Dieses Objekt teilen"** aktivieren, werden diese Notizen für andere Benutzer auf dem Server sichtbar.

**Änderungen speichern**

* **Einzel-Update:** Klicken Sie auf die Schaltfläche **Aktualisieren** in der Box eines Objekts, um Änderungen nur für dieses Ziel zu speichern.
* **Stapel-Update:** Klicken Sie auf die große Schaltfläche **Objekte aktualisieren** ganz unten, um Änderungen in der gesamten Liste zu speichern.
* **Löschen:** Aktivieren Sie das **Lösch**-Kästchen (oder verwenden Sie die Stapel-Löschen-Schaltfläche), um ein Objekt dauerhaft zu entfernen.
