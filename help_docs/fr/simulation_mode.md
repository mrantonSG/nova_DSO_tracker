
# Mode Simulation

Le Mode Simulation vous permet de planifier des sessions d'imagerie pour des dates futures. Au lieu d'afficher le ciel comme il est "maintenant" (temps réel), le tableau de bord calcule les positions des objets, les phases de lune et les heures d'obscurité pour la date que vous sélectionnez.

### Comment l'utiliser
1.  **Basculer l'interrupteur :** Cliquez sur l'interrupteur dans la barre de statut pour activer le Mode Simulation.
2.  **Sélectionner une date :** Cliquez sur le champ de saisie de date pour choisir votre date de planification cible.

### Qu'est-ce qui change ?
* **Calculs :** Les altitudes, azimuts et temps de transit sont recalculés pour la nuit simulée.
* **Indicateurs visuels :** L'arrière-plan de la barre de statut devient **rouge** et un badge "Simulé" apparaît à côté du titre du Tableau de bord pour vous rappeler que les données ne sont pas en temps réel.
* **Graphiques :** Les graphiques d'altitude afficheront la courbe pour la nuit sélectionnée.
* **Inspiration :** Les mini-graphiques dans le modal Inspiration refléteront également la date simulée.

### Notes importantes
* **Heure du jour :** La simulation préserve votre *heure du jour* d'horloge murale actuelle. S'il est 14h00 maintenant, le tableau de bord affiche les positions pour 14h00 à la date simulée.
* **Cache :** Changer de date actualise automatiquement le cache de calcul pour cette date spécifique.
