
### Configuration des emplacements

L'onglet **Emplacements** est l'endroit où vous définissez vos sites d'observation. Qu'il s'agisse de votre jardin, d'un site au ciel sombre ou d'un observatoire distant, les ajouter ici permet à **Nova App** de calculer des heures de lever, de coucher et de transit précises pour votre position spécifique.

#### Ajouter un nouvel emplacement
Pour ajouter un site, remplissez les champs en haut de la page :

  * **Nom :** Un nom convivial pour le site (par ex. "Maison", "Site Désert").
  * **Lat / Lon :** Votre latitude et longitude en degrés décimaux.
  * **Fuseau horaire :** Sélectionnez le bon fuseau horaire pour vous assurer que les heures de vos graphiques correspondent à votre horloge locale.
  * **Actif :** Cochez ceci pour rendre l'emplacement disponible à la sélection dans le tableau de bord principal.

#### Gérer les emplacements
Vous pouvez modifier les emplacements existants directement dans la liste sous le bloc "Ajouter".

  * **Mettre à jour les détails :** Changez les coordonnées ou les fuseaux horaires à tout moment. N'oubliez pas de cliquer sur **Mettre à jour les emplacements** en bas pour enregistrer les modifications.
  * **Statut actif :** Vous pouvez activer ou désactiver les emplacements.
    * *Note : Il y a une limite de **5 emplacements actifs**.*
    * **Important :** Si vous n'utilisez plus un site, **décochez "Actif" au lieu de le supprimer**. Les emplacements inactifs restent disponibles pour vos entrées de **Journal** historiques, garantissant que vos archives restent précises.
  * **Supprimer :** Cochez la case **Suppr** et cliquez sur **Mettre à jour les emplacements** pour retirer définitivement un site. *Avertissement : Cela supprime l'emplacement entièrement.*

#### Masque d'Horizon
Ce champ optionnel vous permet de définir les obstructions (comme les arbres ou les bâtiments) pour empêcher l'application de suggérer des cibles cachées derrière elles.

  * **Pour plus de détails sur le formatage de vos coordonnées de masque**, veuillez cliquer sur le petit badge d'aide **[?]** à côté du label "Masque d'Horizon".
