
### Aide : Gestion du contenu Inspiration

La section **Contenu Inspiration** vous permet d'ajouter des images et des résumés descriptifs à vos objets. Ces détails remplissent les tuiles dans l'**onglet Inspiration** de votre tableau de bord principal, vous donnant un guide visuel de vos cibles.

Vous pouvez modifier ces champs lors de l'**Ajout d'un nouvel objet** ou en mettant à jour un existant dans **Gérer mes objets**.

#### Paramètres d'image

Ces champs déterminent l'image d'arrière-plan affichée sur la tuile Inspiration.

* **URL de l'image :** Collez le lien direct vers le fichier image ici.
* *Astuce :* L'URL devrait généralement se terminer par `.jpg`, `.png` ou `.webp`. Assurez-vous que c'est un lien direct vers l'image, pas la page web contenant l'image.

* **Crédit image :** Entrez le nom du photographe, de l'observatoire ou de l'agence (par ex. "NASA", "Jean Dupont", "DSS2"). Ce texte apparaît comme un badge sur l'image.
* **Lien source image :** Collez l'URL du site web où vous avez trouvé l'image (par ex. la page APOD ou le post Astrobin). Cela permet aux utilisateurs de cliquer pour voir la source originale.

#### Paramètres de description

Ces champs fournissent le résumé textuel superposé à la tuile.

* **Texte de description :** Entrez un résumé court et engageant de l'objet.
* *Astuce :* Restez bref (1-3 phrases) pour que cela s'insère proprement sur la carte.

* **Crédit texte :** Attribuez la source de la description (par ex. "Wikipédia", "The Sky Live").
* **Lien source texte :** Collez l'URL d'où provient le texte de description.

#### Conseils rapides

* **Laisser vide ?** Si vous laissez l'URL de l'image vide, Nova App tentera de charger automatiquement une image DSS2 (Digitized Sky Survey) si des coordonnées sont disponibles.
* **https :** Utilisez toujours des liens sécurisés (`https://`) pour vous assurer que les images se chargent correctement dans tous les navigateurs.
