
#### Gérer mes objets

C'est la salle de contrôle centrale de votre bibliothèque de cibles. Ici vous pouvez affiner les détails des objets, organiser vos projets et garder votre base de données propre.

**Filtrer votre bibliothèque**

Si vous avez une grande bibliothèque, utilisez la barre de filtre en haut pour trouver des cibles spécifiques rapidement. Vous pouvez filtrer par :

* **ID ou Nom :** (par ex. "M42" ou "Orion")
* **Constellation ou Type :** (par ex. "Cyg" ou "Nébuleuse")
* **Statut :** Filtrez par objets **Activés** ou **Désactivés**, ou voyez quels éléments sont **Partagés** ou **Importés**.
* **Source :** Trouvez des objets de catalogues spécifiques (par ex. "messier"). Si vous voulez sélectionner spécifiquement une seule source, utilisez "=messier".

**Actions de sélection (Opérations groupées)**

Utilisez les cases à cocher pour sélectionner plusieurs objets, puis utilisez les boutons d'action :

* **Activer / Désactiver :** Basculez le statut actif des objets sélectionnés. Les objets désactivés sont ignorés pendant les calculs pour de meilleures performances.
* **Récupérer les détails :** Interrogez les catalogues externes pour remplir les données manquantes pour les objets sélectionnés. Cela met à jour les champs **Type**, **Magnitude**, **Taille**, **Brillance de surface** et **Constellation** qui sont vides ou affichent "N/A".
* **Supprimer :** Retirez définitivement les objets sélectionnés. Les objets avec des entrées de Journal ou des Projets ne peuvent pas être supprimés tant que ceux-ci ne sont pas retirés d'abord.

**Désactiver des objets (Performance)**

Pour garder Nova App rapide, vous pouvez **Désactiver** les objets que vous n'imageriez pas actuellement mais que vous ne voulez pas supprimer.

* **Pourquoi désactiver ?** Les objets désactivés sont ignorés pendant les calculs intensifs (comme le Tableau de bord et Perspectives), augmentant significativement les performances logicielles.
* **Sécurisé pour le Journal :** Vous pouvez désactiver un objet sans perdre de données. Vos entrées de **Journal** existantes et journaux pour cet objet resteront entièrement fonctionnels et visibles.
* **Comment faire :** Vous pouvez basculer le statut **Activé/Désactivé** sur un seul objet, ou utiliser la barre **Actions de sélection** en haut pour Activer ou Désactiver plusieurs objets à la fois.

**Modifier les données d'objet**

Vous pouvez modifier la plupart des champs directement dans la liste sans quitter la page :

* **Coordonnées & Statistiques :** Les mises à jour de RA/DEC, Magnitude, Taille ou Brillance de surface sont utiles si vous voulez affiner la précision de position ou ajouter des données manquantes.
* **Projet actif :** Cochez cette case pour marquer l'objet comme priorité. Les objets "Actifs" reçoivent une mise en évidence spéciale dans l'outil de prévision **Perspectives**.

**Notes & Partage**

* **Notes privées :** Utilisez cet espace pour des idées de recadrage personnelles, des plans d'exposition ou des rappels. Ceux-ci ne sont jamais partagés.
* **Notes partagées :** Si vous cochez **"Partager cet objet"**, ces notes seront visibles pour les autres utilisateurs sur le serveur.

**Enregistrer les modifications**

* **Mise à jour unique :** Cliquez sur le bouton **Mettre à jour** dans la boîte d'un objet pour enregistrer les modifications uniquement pour cette cible.
* **Mise à jour groupée :** Cliquez sur le grand bouton **Mettre à jour les objets** tout en bas pour enregistrer les modifications dans toute la liste.
* **Suppression :** Cochez la case **Suppr** (ou utilisez le bouton Suppression groupée) pour retirer un objet définitivement.
