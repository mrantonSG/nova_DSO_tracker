
#### Ajouter un nouvel objet

Utilisez cet onglet pour développer votre bibliothèque. **Nova App** vous aide à automatiser la saisie de données en utilisant des bases de données astronomiques professionnelles.

**Méthode 1 : Recherche automatique (Recommandée)**

1.  Entrez un **ID d'objet** (par ex. "M31", "NGC 7000", "Gaia 123...").
2.  Cliquez sur **Rechercher**.
3.  Nova vérifiera d'abord votre bibliothèque locale. S'il n'est pas trouvé, il interroge la base de données **SIMBAD**.
4.  S'il est trouvé, les champs (RA, DEC, Magnitude, Taille, Type) seront automatiquement remplis.
5.  Cliquez sur **Confirmer l'ajout** pour l'enregistrer dans votre bibliothèque. (vous pouvez aussi cliquer sur **éditer** si vous voulez changer le contenu rempli automatiquement, par ex. le nom)

**Méthode 2 : Saisie manuelle**

Si vous voulez ajouter une cible non trouvée par la recherche automatique :

1.  Remplissez l'**ID d'objet** et le **Nom**.
2.  Tapez manuellement les coordonnées **RA** et **DEC**.
3.  Remplissez les statistiques optionnelles comme la Magnitude ou la Constellation.
4.  Cliquez sur **Confirmer l'ajout**.

**Ajouter des notes immédiatement**

Vous pouvez ajouter vos **Notes privées** (par ex. "Plan mosaïque") ou **Notes partagées** directement ici avant d'enregistrer, pour que votre objet soit prêt à l'emploi dès qu'il est ajouté.
