
#### Importer depuis le catalogue

Cet onglet vous permet d'importer en lot des collections d'objets organisées fournies par le serveur. C'est le moyen le plus rapide de peupler une nouvelle bibliothèque.

**Packs de catalogue**

Le tableau affiche les "Packs" disponibles (par ex. "Objets Messier", "Catalogue Caldwell", "Catalogue Sharpless"). Chaque entrée affiche :

* **Nom & Description :** Ce que contient le pack.
* **Tags :** Catégories rapides (par ex. "Nébuleuse", "Débutant").
* **Nombre :** Combien d'objets sont dans le pack.

**Comment fonctionne l'importation**

Cliquez sur le bouton **Importer** à côté d'un pack pour commencer.

* **Nouveaux objets :** Les objets que vous n'avez pas actuellement seront ajoutés à votre bibliothèque.
* **Objets existants (Mises à jour) :** Si vous avez déjà un objet (par ex. "M42"), Nova va **mettre à jour** ses champs "Inspiration" (URL de l'image, Description, Crédits) avec les données de haute qualité du catalogue.
* **Sécurisé pour les données personnalisées :** Vos Notes de projet personnelles, Statut actif et Recadrages sauvegardés ne sont **jamais écrasés**. L'importation améliore uniquement les données statiques du catalogue.
