# 了解天顶SQM

**SQM**（天空质量仪）以**mag/arcsec²**为单位测量天顶处天空的暗度。数值越高表示天空越暗——真正黑暗的地点约为21.5–22，而郊区天空可能在19–20之间。

## Bortle与SQM

如果你将天顶SQM留空，Nova会根据你的**波特尔等级**设置推算一个名义SQM值。这对大多数规划用途来说是合理的估算。

如果你有来自测量仪器或*Clear Outside*等应用的实测SQM值，请在此输入。Nova将使用你的实测值而非波特尔估算值——这能提高极限星等计算和AI排名评分的准确性。

## 各波特尔等级的典型值

| 波特尔 | 天空类型 | 典型SQM |
|--------|----------|---------|
| 1 | 真正黑暗 | ≥ 21.9 |
| 3 | 乡村 | ~21.5 |
| 5 | 郊区 | ~20.4 |
| 7 | 郊区/城市 | ~19.1 |
| 9 | 市中心 | ≤ 18.0 |

## 提示

- 在晴朗无月的夜晚于天顶进行一次测量即可。
- SQM会随湿度、烟雾和季节性气辉而变化——无需追求极高精度，误差在0.2 mag以内就足够准确。
- 如果你从多个地点观测，每个地点可以有自己的SQM值。
