# 校准星

在开始导星之前，PHD2 和 ASIAIR 需要**校准**导星相机的轴，通过移动赤道仪的 RA 和 Dec 并测量产生的星体运动。为了获得最准确的校准，您的导星应满足两个条件:

1. **靠近天赤道**: 赤纬在 0° 的 **±20°** 范围内，此时 RA/Dec 运动最正交
2. **靠近子午线**: 时角在 **±1.5 小时** 以内，可将赤道仪锥轴误差和 Dec 回差影响降至最低

## 此组件的功能

此组件会为您的位置和所选日期找到满足这两个条件的**最佳可用亮星**，然后显示该星在校准可用区域内的时间窗口。

## 使用方法

1. 在图表选项卡中将**日期**设置为您计划的观测夜晚
2. 组件会显示推荐的校准星及其 **RA/Dec 坐标**
3. **校准窗口**显示该星何时处于最佳区域内
4. 在 ASIAIR 中，在开始导星校准之前，**转到显示的 RA/Dec 位置**
5. 完成校准后，再转回您的成像目标

## 提示

- 如果未找到星体，请尝试选择其他日期或检查您的位置设置是否正确
- 刷新按钮 (↻) 会重新对当前日期运行星体搜索
- 较亮的星（星等较低）更受青睐，因为导星检测更可靠
