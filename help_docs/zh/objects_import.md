
#### 从目录导入此选项卡可批量导入服务器提供的精选天体集合。这是快速填充新星库的最便捷方式。**目录包**表格显示可用"包"（如"梅西耶天体"、"考德威尔星表"、"夏普莱斯星表"）。每项显示：

* **名称与描述：** 说明包内包含的内容。* **标签：** 快速分类（例如"星云"、"初学者"）。* **数量：** 包内包含的天体数量。**导入流程** 点击包旁的**导入**按钮开始操作。

* **新增天体：**当前未拥有的天体将添加至您的图库。* **已存在天体（更新）：**若您已拥有某天体（如"M42"），Nova将使用星表的高质量数据**更新**其"灵感"字段（图像URL、描述、来源信息）。
* **自定义数据安全保障：** 您的项目笔记、活动状态和保存的构图设置**绝不会被覆盖**。导入仅增强静态目录数据。
