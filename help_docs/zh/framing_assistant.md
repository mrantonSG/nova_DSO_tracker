
#### 构图助手**构图助手**是一款强大的视觉工具，可让您精准预览目标在相机中的呈现效果。它将设备视场（FOV）叠加至专业天空测绘图像之上。

**入门指南**1. **选择设备：**使用顶部下拉菜单选择预设的设备配置文件。屏幕上的矩形区域代表传感器视场范围。
2. **移动与居中：** * **点击拖拽：** 默认开启**锁定视场**功能。此时传感器矩形将固定在屏幕中央，拖动鼠标可移动其*后方*的天空区域，模拟望远镜在天空中移动以构图的目标过程。
    * **微调控制：**（关闭视场锁定）使用工具栏中的箭头按钮（↑ ↓ ← →）或键盘方向键，以1角分为单位进行精细调整。* **重新居中：**点击"对准目标居中"可将视图快速恢复至目标的星表坐标位置。

**构图控制** * **旋转：** 通过滑块旋转摄像机角度（0-360°）。 * **ASIAIR兼容性：** 此处显示的角度与**ASIAIR**应用中的"取景辅助"角度完全对应，便于在实地复现您的计划。
    * *技巧：* 点击滑块旁的角度文字可快速重置为0°。 * **地球同步轨道带：** 勾选此框可叠加地球同步卫星轨道带（显示为紫色虚线）。 * **用途：** 地球同步卫星相对于地面保持静止，这意味着当望远镜追踪星体时，它们会在图像中划出轨迹。 使用此叠加层可确保取景框不直接覆盖这条"卫星高速公路"。* **精度：**根据您所在纬度自动计算并校正视差位置。* **观测模式：**通过"Survey"下拉菜单切换背景图像源。
    * **DSS2（彩色）：** 通用光学观测效果优良。* **H-alpha：** 极佳呈现暗弱星云结构。* **混合模式：** 通过"混合方式"下拉菜单与透明度滑块，可在基础彩色图像上叠加第二张观测图（如H-alpha）。此功能在保留恒星色彩的同时，可揭示隐藏细节。

**拼接规划器** 若目标区域超出单帧范围，请使用工具栏中的**拼接**功能：1. 设置**列数**与**行数**（如2x1用于宽幅全景）。2. 调整**重叠百分比**（默认10%）。
3. **复制计划：**点击"复制计划(CSV)"生成兼容**ASIAIR**或**N.I.N.A.**等采集软件的坐标列表。

**保存工作** * **保存构图：**点击此项将当前支架位置、旋转角度及中心坐标存储至数据库。下次访问该目标时，自定义构图将自动恢复。 * **锁定视场：**默认勾选。取消勾选可解锁传感器矩形框，允许在静态天图上拖动矩形框位置。
