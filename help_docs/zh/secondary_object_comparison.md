# 次要目标比较通过**添加目标**下拉菜单，您可在图表视图中将另一个目标的高度曲线与主要目标并列比较。## 目标排序规则下拉菜单中的目标按**总可观测时长**排序（最长者优先）。此功能可帮助您快速识别今晚可见时间最长的活跃目标。

## 筛选条件 仅满足以下**全部**条件的对象才会显示在列表中：1. **活跃项目** — 该对象必须标记为活跃项目（在注释与构图选项卡中勾选复选框）
2. **今夜可观测** — 天文黑暗时段内，目标必须位于您的地平线遮罩上方 3. **有效坐标** — 目标必须定义有赤经(RA)和赤纬坐标 4. **非主目标** — 您当前正在观测的目标将从列表中排除 ## 限制条件

- 仅显示观测时长排名前**20位**的天体 - 观测时长为**0分钟**的天体将被排除 ## 显示方式 - 次要天体的高度以**实心洋红色线条**标注 - 次要天体不显示方位角线 （以减少星图杂乱） - 选择**无**将移除次要比较并恢复默认视图 ## 技巧 - 通过比较多个目标规划观测行程 - 特别适用于选择不同时间过境的天体 - 比较激活时页面标题将更新为显示两个目标名称
