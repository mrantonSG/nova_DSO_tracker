
# DSO 高度图

图表显示了以当地午夜为中心的 24 小时窗口，让您全面了解今晚目标的观测时间。

## 图表线条

**目标高度**: 蓝色线条，显示目标在白天和夜间相对于地平线的高度。

**月球高度**: 黄色线条，显示月球位置。当月球较高且靠近您的目标时，天背景会增加。

**目标方位角 / 月球方位角**: 右侧坐标轴上的虚线，显示各目标的罗盘方向（0–360°）。可用于检查是否有障碍物或光源遮挡。

## 叠加层

**地平线掩膜**: 灰色阴影区域代表您配置的地形掩膜。高度曲线在此区域内的任何部分都被当地地形或障碍物遮挡。

**地平线**: 0° 处的水平线，标记数学地平线。

**Skyglow 底面**: 琥珀色阴影区域显示天空背景在每个罗盘方向达到站点 SQM 阈值时的最低高度。当目标高于此底面时，处于相对干净的天空中。当目标低于此底面时，来自该方向的光污染变得显著。形状随方位角变化: 朝向城市或城镇的方向会显示更高的底面，而黑暗方向则较低。在位置设置中配置站点的 Bortle 等级或 SQM Zenith 值以获得准确结果。

## 天气栏

顶部的彩色条按小时显示预测的云量覆盖和视宁度条件。悬停在某一段上可查看详细信息。

## 太阳事件

垂直线标记日落、天文暮光、天文晨光和日出。最佳观测窗口通常在天文暮光与天文晨光之间。

## 日期和视图控制

使用底部的日/月/年控件导航到特定日期。**模拟模式**（标题栏中的时钟图标）让您预览任何夜晚，仿佛就是今晚。
