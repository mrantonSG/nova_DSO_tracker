#### 主仪表盘这是您的任务控制中心。主仪表盘实时呈现目标星图概览，根据您当前所在位置与时间动态计算生成。其核心设计旨在解答：*"此刻最适合拍摄什么天体？"*

**可见性说明** 默认情况下，从您当前位置几何上无法观测的天体（即永远不会升至您设置的地平线阈值以上）将被**隐藏**以保持列表简洁。若您通过名称或ID主动搜索，这些天体将立即重新显示。**数据列** * **高度/方位角：** 当前实时位置。
* **23:00时位置：** 今晚23:00的观测位置，助您规划核心成像时段。* **趋势：** 显示天体正升起（↑）或落下（↓）。* **最高仰角：** 天体今晚达到的最高点。* **可观测时长：** 天体位于您设置的地平线阈值以上区域的总分钟数。

**高级筛选** 标题下方筛选行功能强大。可使用特殊运算符优化列表：* **文本搜索：** 正常输入查找匹配项（如`M31`、`星云`）。注意：搜索特定天体会覆盖"隐藏不可见天体"设置。
* **数值比较：** * `>50`：匹配大于50的数值。 * `<20`：匹配小于20的数值。 * `>=` / `<=`：大于/小于或等于。 * **区间筛选（AND逻辑）：**组合运算符可查找特定区间内的数值。
* 示例：在*方位角*列输入`>140 <300`可查找当前位于南天（140°至300°之间）的天体。* **排除（NOT逻辑）：**以`!`开头排除项目。* 示例：在*类型*列输入`!Galaxy`可隐藏所有星系。
* 示例：在星座列输入`!Cyg`可隐藏天鹅座目标。* **多条件（OR逻辑）：** 用逗号分隔条件。* 示例：在目标列输入`M31, M33, M42`仅显示这三个目标。* 示例：在类型列输入`Nebula, Cluster`可同时显示星云与星团。

**保存视图** 创建实用筛选集（如"南天高空星系"）后，点击"保存视图"下拉框旁的**保存**按钮。可为该视图命名并随时调用。**视觉探索** **灵感**标签页提供可视化浏览潜在目标的方式。 取代数据表格呈现： * **智能推荐：** 应用自动高亮"精选目标"——当前位置优越（高仰角）且当晚观测时长充足的天体。 * **视觉卡片：** 每个目标以卡片形式展示，包含图像、描述及关键参数（最大仰角、观测时长），一目了然。
* **交互式详情：**点击卡片可查看完整信息或直接跳转至星图。 **标签页功能** * **位置：**实时坐标与可见状态。 * **属性：**恒星等、尺寸、星座等静态数据。
* **观测预告：**长期预报显示适合拍摄活跃项目的最佳观测夜。* **热力图：**可视化年度日历标注天体可见时段。* **灵感库：**当前可见目标的图像画廊，含图片与简介。* **日志：**快速访问所有记录观测会话的列表。
