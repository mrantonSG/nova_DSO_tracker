# 模拟模式模拟模式允许您为未来日期规划成像观测。仪表盘不会显示"当前"（实时）的天空状态，而是根据您选择的日期计算天体位置、月相及黑暗时段。### 使用方法1. **切换开关：**点击状态栏中的开关按钮启用模拟模式。
2. **选择日期：**点击日期输入框选择目标规划日期。 ### 哪些内容会变化？ * **计算结果：** 模拟夜间的高度角、方位角及过境时间将重新计算。
* **视觉标识：**状态栏背景转为**红色**，仪表板标题旁出现"模拟"标识，提醒您数据非实时更新。* **图表：**仰角图将显示所选夜间的曲线。* **灵感：**灵感模态窗口中的迷你图表也将反映模拟日期。

### 重要说明 * **时间点：** 模拟功能将保留您当前的墙钟时间。若当前为下午2:00，仪表盘将显示模拟日期下午2:00的星体位置。 * **缓存机制：** 切换日期将自动刷新该特定日期的计算缓存。
