
#### 添加新天体使用此选项卡扩展您的天体库。**Nova应用**可通过专业天文数据库实现数据录入自动化。**方法1：自动搜索（推荐）**

1. 输入**天体ID**（例如"M31"、"NGC 7000"、"Gaia 123..."）。2. 点击**搜索**。3. Nova将优先检索本地库，若未找到则查询**SIMBAD**数据库。
4. 若检索成功，相关字段（RA、赤纬、星等、尺寸、类型）将自动填充。5. 点击**确认添加**将其保存至您的星表库。（若需修改自动填充内容如名称，也可点击**编辑**）

**方法二：手动输入** 若需添加自动检索未找到的目标：1. 填写**天体ID**和**名称**。2. 手动输入**RA**和**赤纬**坐标。3. 可选填写星等、星座等参数。
4. 点击**确认添加**。**即时添加备注** 保存前可在此直接添加**私人备注**（如"拼接计划"）或**共享备注**，确保目标添加完成即可立即使用。
