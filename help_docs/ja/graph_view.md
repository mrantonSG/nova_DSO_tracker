
# DSO 高度グラフ

グラフは現地時刻の真夜中を中心に24時間表示し、今夜ターゲットが観測可能かを完全に把握できます。

## チャートライン

**天体高度**: 青い線は、一日中および夜を通じて地平線からのターゲットの高度を示します。

**月の高度**: 黄色い線は月の位置を示します。月が高くターゲットに近い場合、天空背景が増加する可能性があります。

**天体方位角 / 月方位角**: 右軸の破線は、各天体のコンパス方向（0–360°）を示します。障害物や光源が邪魔になっていないか確認するのに役立ちます。

## オーバーレイ

**ホライゾンマスク**: 灰色の shaded エリアは設定した地形マスクを表します。このエリア内の天体高度曲線の部分は、当地の地形や障害物によってブロックされます。

**ホライゾン**: 数学的な地平線を示す0°の水平線。

**Skyglow フロア**: アンバー色の shaded エリアは、各コンパス方向で天空背景がサイトの SQM 閾値に達する最低高度を示します。天体がこのフロアより高い場合、比較的クリーンな空にあります。低い場合、その方向からの光汚染が重要になります。形状は方位角によって変化し: 都市や町への方向は暗い方向よりも高いフロアを示します。正確な結果を得るため、場所設定でサイトの Bortle 等級または SQM Zenith 値を設定してください。

## 天気バー

上部のカラーバーは、雲量とシーイング条件を時間別に予測表示します。セグメントにカーソルを合わせると詳細が表示されます。

## 太陽イベント

垂直線は、日没、天文薄明、天文夜明け、日の出を示します。最適な撮影ウィンドウは通常、天文薄明から天文夜明けの間です。

## 日付と表示コントロール

下部の日/月/年のコントロールを使用して特定の日付に移動できます。**シミュレーションモード**（ヘッダーの時計アイコン）は、今夜であるかのように任意の夜をプレビューできます。
