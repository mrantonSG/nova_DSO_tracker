
#### カタログからインポート

このタブでは、サーバーが提供するキュレートされたオブジェクトコレクションを一括インポートできます。これは新しいライブラリをpopulateする最速の方法です。

**カタログパック**

テーブルには利用可能な「パック」が表示されます（例：「メシエオブジェクト」、「コールドウェルカタログ」、「シャープレスカタログ」）。各エントリには：

* **名前と説明：** パックに含まれるもの。
* **タグ：** クイックカテゴリ（例：「星雲」、「初心者」）。
* **数：** パック内のオブジェクト数。

**インポートの動作**

パックの横にある**インポート**ボタンをクリックして開始します。

* **新しいオブジェクト：** 現在持っていないオブジェクトがライブラリに追加されます。
* **既存のオブジェクト（更新）：** 既にオブジェクト（例：「M42」）を持っている場合、Novaはカタログからの高品質データで「インスピレーション」フィールド（画像URL、説明、クレジット）を**更新**します。
* **カスタムデータは安全：** 個人的なプロジェクトメモ、アクティブステータス、保存されたフレーミングは**決して上書きされません**。インポートは静的なカタログデータのみを強化します。
