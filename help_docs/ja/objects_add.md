
#### 新しいオブジェクトを追加

このタブを使用してライブラリを拡張します。**Nova App**はプロフェッショナルな天体データベースを使用してデータ入力を自動化するのに役立ちます。

**方法1：自動検索（推奨）**

1.  **オブジェクトID**を入力します（例：「M31」、「NGC 7000」、「Gaia 123...」）。
2.  **検索**をクリックします。
3.  Novaはまずローカルライブラリをチェックします。見つからない場合、**SIMBAD**データベースにクエリを送信します。
4.  見つかった場合、フィールド（RA、DEC、等級、サイズ、タイプ）が自動入力されます。
5.  **追加を確認**をクリックしてライブラリに保存します。（自動入力されたコンテンツを変更したい場合、例えば名前など、**編集**をクリックすることもできます）

**方法2：手動入力**

自動検索で見つからないターゲットを追加したい場合：

1.  **オブジェクトID**と**名前**を入力します。
2.  **RA**と**DEC**座標を手動で入力します。
3.  等級や星座などのオプション統計を入力します。
4.  **追加を確認**をクリックします。

**すぐにメモを追加**

保存する前に**プライベートメモ**（例：「モザイクプラン」）や**共有メモ**をここに追加できるため、オブジェクトは追加されるとすぐに使用可能になります。
