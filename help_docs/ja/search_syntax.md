
#### メインダッシュボード

これはミッションコントロールです。メインダッシュボードは、現在の場所と時間に対して計算されたターゲットライブラリのリアルタイム概要を提供します。*"今何を撮影するのがベストか？"*という質問に答えるように設計されています。

**可視性ノート**
デフォルトでは、現在の場所から幾何学的に見ることが不可能なオブジェクト（つまり、設定された地平線しきい値より上に上がらないもの）はリストをきれいに保つために**非表示**になります。これらのオブジェクトは、名前またはIDで明示的に検索するとすぐに再表示されます。

**データカラム**

* **高度/方位角：** 現在のリアルタイム位置。
* **23時：** 今夜23時の位置。コアとなる撮影時間の計画に役立ちます。
* **トレンド：** オブジェクトが上昇中（↑）または下降中（↓）かを表示。
* **最大高度：** 今夜オブジェクトが到達する最高点。
* **可観測時間：** オブジェクトが設定された地平線制限より上にある合計分数。

**高度なフィルタリング**

ヘッダーの下のフィルター行は強力です。特殊な演算子を使用してリストを絞り込めます：

* **テキスト検索：** 通常通り入力して一致を検索（例：`M31`、`Nebula`）。特定のオブジェクトを検索すると「非表示のオブジェクトを隠す」設定が上書きされることに注意してください。
* **数値比較：**
* `>50`：50より大きい値に一致。
* `<20`：20より小さい値に一致。
* `>=` / `<=`：以上/以下。
* **範囲（AND ロジック）：** 演算子を組み合わせて特定のウィンドウ内の値を検索。
* 例：*方位角*カラムで`>140 <300`は、現在南方の空にあるオブジェクト（140°〜300°の間）を検索。
* **除外（NOT ロジック）：** `!`で始めてアイテムを除外。
* 例：*タイプ*カラムで`!Galaxy`はすべての銀河を非表示。
* 例：*星座*で`!Cyg`ははくちょう座のターゲットを非表示。
* **複数用語（OR ロジック）：** 用語をカンマで区切ります。
* 例：*オブジェクト*で`M31, M33, M42`はこれら3つのターゲットのみを表示。
* 例：*タイプ*で`Nebula, Cluster`は星雲と星団の両方を表示。

**保存されたビュー**

便利なフィルタセット（例：「南方で高い銀河」）を作成したら、「保存されたビュー」ドロップダウンの横にある**保存**ボタンをクリックします。このビューに名前を付け、後で即座に呼び出せます。

**ビジュアルディスカバリー**

**インスピレーション**タブは、潜在的なターゲットを閲覧するグラフィカルな方法を提供します。データテーブルの代わりに：

* **スマート提案：** アプリは自動的に「トップピック」をハイライト - 現在よく位置している（高高度）オブジェクトで、夜間の長い可観測時間を持つもの。
* **ビジュアルカード：** 各ターゲットはカードとして表示され、画像、説明、主要な統計（最大高度、期間）が一目でわかります。
* **インタラクティブな詳細：** 任意のカードをクリックして完全な詳細を表示するか、そのチャートに直接ジャンプします。

**タブ**

* **位置：** リアルタイム座標と可視性。
* **プロパティ：** 等級、サイズ、星座などの静的データ。
* **見通し：** アクティブプロジェクトを撮影するのに最適な夜を示す長期予測。
* **ヒートマップ：** オブジェクトが可視可能な時期を示すビジュアル年間カレンダー。
* **インスピレーション：** 画像と要約付きの現在可視可能なターゲットのビジュアルギャラリー。
* **ジャーナル：** 記録されたすべてのセッションのクイックアクセスリスト。
