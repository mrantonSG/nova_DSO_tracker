
# シミュレーションモード

シミュレーションモードでは、将来の日付の撮影セッションを計画できます。「今」の空を表示する（リアルタイム）代わりに、ダッシュボードは選択した日付のオブジェクト位置、月相、暗さの時間を計算します。

### 使用方法
1.  **スイッチを切り替え：** ステータスバーのトグルスイッチをクリックしてシミュレーションモードを有効にします。
2.  **日付を選択：** 日付入力フィールドをクリックして、計画対象の日付を選択します。

### 何が変わるか？
* **計算：** 高度、方位角、南中時刻はシミュレートされた夜に対して再計算されます。
* **ビジュアルインジケーター：** ステータスバーの背景が**赤**になり、ダッシュボードタイトルの横に「シミュレート済み」バッジが表示され、データがリアルタイムではないことを思い出させます。
* **グラフ：** 高度チャートは選択した夜の曲線を表示します。
* **インスピレーション：** インスピレーションモーダルのミニグラフもシミュレートされた日付を反映します。

### 重要な注意点
* **時刻：** シミュレーションは現在のウォールクロックの*時刻*を保持します。現在14:00の場合、ダッシュボードはシミュレートされた日付の14:00の位置を表示します。
* **キャッシュ：** 日付を切り替えると、その特定の日付の計算キャッシュが自動的に更新されます。
