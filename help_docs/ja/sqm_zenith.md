# 天頂SQMについて

**SQM**（スカイクオリティメーター）は、天頂における空の暗さを**mag/arcsec²**単位で測定する。値が高いほど空が暗いことを意味し、本当に暗い観測地では21.5〜22程度、郊外の空では19〜20程度になる。

## BortleとSQM

天頂SQMを空白のままにすると、Novaは**ボートル等級**の設定から標準的なマッピングを使って名目SQM値を導出する。これはほとんどの計画用途において妥当な推定値である。

測定器や*Clear Outside*などのアプリから実測SQM値がある場合は、ここに入力してください。Novaはボートル推定値の代わりにあなたの実測値を使用する——これにより、限界等級の計算とAIランカーのスコアリングの精度が向上する。

## ボートル等級別の典型的な値

| ボートル | 空のタイプ | 典型的なSQM |
|----------|-----------|------------|
| 1 | 真の暗黒 | ≥ 21.9 |
| 3 | 農村部 | ~21.5 |
| 5 | 郊外 | ~20.4 |
| 7 | 郊外/都市 | ~19.1 |
| 9 | 都市中心部 | ≤ 18.0 |

## ヒント

- 晴れた月のない夜に天頂で一度測定するだけで十分である。
- SQMは湿度、煙、季節的なエアグロウによって変化する——細かい精度は気にしなくて大丈夫である。実際の値から0.2 mag以内であれば十分正確である。
- 複数の観測地から撮影する場合、各観測地に独自のSQM値を設定できる。
