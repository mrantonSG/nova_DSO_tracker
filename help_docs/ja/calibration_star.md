# キャリブレーション星

ガイドを開始する前に、PHD2 と ASIAIR は架台を RA と Dec に動かし、その結果生じる星の動きを測定することで、ガイドカメラの軸を**キャリブレーション**する必要があります。最も正確なキャリブレーションを行うには、ガイド星が次の2つの条件を満たす必要があります:

1. **天の赤道付近**: 赤緯が 0° の **±20°** 以内。RA/Dec の動きが最も直交する位置です
2. **子午線付近**: 時角が **±1.5 時間** 以内。架台のコーンエラーと Dec バックラッシュの影響を最小限に抑えます

## このウィジェットの機能

このウィジェットは、あなたの場所と選択した日付において両方の条件を満たす**最適な明るい星**を見つけ、その星が使用可能なキャリブレーションゾーン内にある時間枠を表示します。

## 使い方

1. チャートタブで**日付**を撮影予定の夜に設定します
2. ウィジェットが推奨されるキャリブレーション星とその **RA/Dec 座標**を表示します
3. **キャリブレーションウィンドウ**は、星が最適なゾーン内にある時間を示します
4. ASIAIR で、ガイドキャリブレーションを開始する前に、**表示された RA/Dec に向けて駆動**します
5. キャリブレーションを完了した後、撮影ターゲットに戻ります

## ヒント

- 星が見つからない場合は、別の日付を選択するか、場所が正しく設定されているか確認してください
- 更新ボタン (↻) を押すと、現在の日付で星の検索を再実行します
- 明るい星（等級が低い星）ほどガイド星の検出が確実なため推奨されます
