
# セカンダリオブジェクト比較

**オブジェクトを追加**ドロップダウンを使用すると、グラフビューでプライマリターゲットと一緒に別のオブジェクトの高度曲線を比較できます。

## オブジェクトのランク付け方法

ドロップダウン内のオブジェクトは**総可観測時間**でソートされています（最長が最初）。これにより、今夜最も長く可視可能なアクティブターゲットを素早く特定できます。

## フィルタリング基準

オブジェクトは以下の**すべて**を満たす場合にのみリストに表示されます：

1. **アクティブプロジェクト** — オブジェクトはアクティブプロジェクトとしてマークされている必要がある（メモとフレーミングタブのチェックボックス）
2. **今夜可観測** — オブジェクトは天文薄明中にホライゾンマスクより上にある必要がある
3. **有効な座標** — オブジェクトはRAとDEC座標が定義されている必要がある
4. **プライマリではない** — 現在表示しているオブジェクトはリストから除外される

## 制限

- 可観測時間による**上位20**オブジェクトのみが表示される
- **0分**の可観測時間を持つオブジェクトは除外される

## 表示

- セカンダリオブジェクトの高度は**実線のマゼンタライン**として表示される
- セカンダリオブジェクトの方位角ラインは描画されない（チャートの混雑を減らすため）
- **なし**を選択するとセカンダリ比較が削除され、デフォルトビューが復元される

## ヒント

- この機能を使用して複数のターゲットを比較して夜を計画
- 異なる時間に南中するオブジェクト間で決定するのに最適
- 比較がアクティブな場合、ページタイトルは両方のオブジェクト名を表示するように更新される
