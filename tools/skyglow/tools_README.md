# Nova Tools

Standalone utilities that are not yet wired into the Flask app but are
tracked here for future integration.

## skyglow/

Directional sky brightness (skyglow horizon) from NASA VIIRS data.
Computes SQM as a function of altitude and azimuth for any location,
using a simplified Garstang (1986) atmospheric scattering model.

See `skyglow/README.md` for setup and usage.
